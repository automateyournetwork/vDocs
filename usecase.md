vDocs - Transform Cisco SD-WAN vManage REST APIs into Business Ready Documents
=====================================

vDocs uses various vManage APIC REST APIs and transforms them, using Jinja2 templates into:
RAW JSON files
RAW YAML files
CSV files
Markdown files
Dynamic HTML pages
Mindmaps
